API ENDPOINTS SUMMARY:
======================

Chat:
  POST   live.chat.send(message)
  GET    live.chat.messages(limit)
  DELETE live.chat.delete(messageId)
  UPDATE live.chat.pinMessage(messageId)
  UPDATE live.chat.unpinMessage(messageId)
  GET    live.chat.pinnedMessages(sessionId)

Donations:
  POST   live.donations.create(amount, message)
  POST   live.donations.confirm(donationId, paymentIntentId)
  GET    live.donations.history(limit)
  GET    live.donations.sessionTop(period)

Reactions:
  POST   live.reactions.add(reactionType)
  GET    live.reactions.counts(timeWindow)
  GET    live.reactions.topCombos(limit)

Polls:
  POST   live.polls.create(question, options, duration)
  POST   live.polls.vote(pollId, optionIndex)
  GET    live.polls.getActive(sessionId)
  UPDATE live.polls.close(pollId)

Leaderboards:
  GET    live.leaderboard.topDonors(period, limit)
  GET    live.leaderboard.topChatters(limit)
  UPDATE live.leaderboard.sync(sessionId)

Notifications:
  GET    live.notifications.getUnread(sessionId)
  UPDATE live.notifications.markRead(notificationId)

Stats:
  GET    live.stats.get(streamerId)
  UPDATE live.stats.update(streamerId, updates)

Social:
  GET    live.social.getLinks(userId)
  UPDATE live.social.updateLinks(links)
  POST   live.social.raid(toStreamerId, viewers)

INSTALLATION STEPS:
===================

1. Create migration:
   drizzle-kit generate && pnpm db:push

2. Add router to server/routers.ts:
   import { liveRouter } from "./routers/liveRouter";
   export const appRouter = router({ live: liveRouter, ... });

3. Set Stripe environment variables:
   STRIPE_SECRET_KEY=sk_test_...
   STRIPE_PUBLISHABLE_KEY=pk_test_...
   STRIPE_WEBHOOK_SECRET=whsec_...

4. Initialize WebSocket in server:
   import { initializeLiveWebSocket } from "./routers/liveWebSocket";
   const wss = new WebSocket.Server({ noServer: true });
   initializeLiveWebSocket(wss);

5. Add webhook endpoint:
   POST /api/webhooks/stripe with webhook handler

6. Wrap components with Stripe Elements provider:
   <Elements stripe={stripePromise}>
     <LiveStreamPage />
   </Elements>

7. Import components in your page:
   import { LiveChat } from "@/components/live/LiveChat";
   import { DonationPanel } from "@/components/live/DonationPanel";
   // ... other components
